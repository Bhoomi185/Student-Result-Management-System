# 🎓 Student Result Management System

## 📖 About
This is a Python-based Student Result Management System developed as a beginner project to practice Python programming concepts.

The project allows users to:
- Add student records
- View all students
- Search for a student
- Delete student records
- Calculate percentage
- Display grades

## 🛠️ Technologies Used
- Python 3
